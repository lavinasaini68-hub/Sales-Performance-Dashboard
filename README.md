# 📊 Sales Performance Dashboard

## 🗂️ Project Overview
An Excel-based sales analytics dashboard analyzing **3,500 orders** 
across **4 regions**, **3 categories** and **10 products** from 2022 to 2024.

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- SUMIF, INDEX MATCH Formulas
- Interactive Charts
- Dynamic Slicers

---

## 📈 Key Insights
| Metric | Value |
|--------|-------|
| Total Revenue | ₹1,06,67,881 |
| Total Orders | 3,500 |
| Top Product | Camera |
| Best Region | West (₹28,44,450) |
| Peak Year | 2023 (₹37,86,592) |
| Top Category | Electronics (50%) |

---

## ✨ Dashboard Features
- 6 Interactive Charts
- Dynamic Slicers for Year, Region and Category
- 5 Structured Sheets
- Data Cleaning and Validation
- KPI Cards with key business metrics

---

## 📋 Sheets Breakdown
| Sheet | Description |
|-------|-------------|

| cleandata | Cleaned data with Unit Price and Sorted Date |
| pivot table | 6 pivot tables with key analysis |
